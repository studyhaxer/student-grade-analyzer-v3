# Student Grade Analyzer V3

A Python command-line application for managing student records using Object-Oriented Programming (OOP) and JSON storage.

## Features

* Add Students
* View Students
* Search Student
* Delete Student
* Find Topper
* Calculate Class Average
* Count Total Students
* Save Students to JSON
* Load Students from JSON
* Input Validation

## Concepts Practiced

* Classes and Objects
* Functions
* Lists and Dictionaries
* File Handling
* JSON Serialization and Deserialization
* Exception Handling
* Menu-Driven Applications
* CRUD Operations

## Technologies

* Python 3

## How to Run

```bash
python3 Student_Grade_Analyzer_V3.py
```
